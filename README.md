# higherOrderFunctions-each


## Introduction

Il est bon de savoir que nous avons **deux formes de fonctions** :

### 1. Déclaration de fonction (Function Declaration)

```javascript
function square(x) {
    return x * x;
}
```

### 2. Expression de fonction (Function Expression)

```javascript
var square = function(x) {
    return x * x;
};
```
